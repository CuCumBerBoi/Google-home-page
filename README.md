# Google Home Page Clone

This project is a visually accurate clone of the Google Home Page, built using HTML, CSS, and Vanilla JavaScript. It mimics the modern light theme design of Google.

## Features

*   **Responsive Design**: The layout adjusts to different screen sizes.
*   **Search Functionality**:
    *   **Google Search**: Redirects the query to the official Google search results.
    *   **I'm Feeling Lucky**: Simulates the "I'm Feeling Lucky" functionality (redirects to search/doodles).
*   **Visual Fidelity**:
    *   Replicates the modern Google search bar with Microphone and Lens icons (visual only).
    *   Includes the specific "Sign In" button hover effects using CSS pseudo-elements.
    *   "Google offered in" section with Thai language option.
*   **Educational**: The source code is heavily commented in **Thai** to explain every line of HTML, CSS, and JavaScript for learning purposes.

## Technologies Used

*   HTML5
*   CSS3 (Flexbox, CSS Transitions, Pseudo-elements)
*   JavaScript (DOM Manipulation, Event Listeners)

## How to Run

1.  Clone this repository or download the files.
2.  Open the folder containing the files.
3.  Double-click `index.html` to open it in your web browser.

## Project Structure

*   `index.html`: The main structure of the webpage.
*   `style.css`: All styling rules (light theme).
*   `script.js`: Logic for search input and buttons.

## Learning

This project was created to study:
*   Web page layouts using Flexbox.
*   CSS styling techniques including `:hover` states, transitions, and `::before` pseudo-elements.
*   Basic JavaScript for handling user input and browser redirection.
*   Git version control basics.
